Litter Picking Robot

  1. Home
  2. Projects

Our team has developed a comprehensive robotics solution for trash detection and handling. Our system comprises a mobile base designed in SolidWorks and exported as URDF, with an integrated Panda arm model obtained from a GitHub repository. To detect trash, we use a Kinect camera, which not only captures RGB images but also provides depth information. We use YOLO to identify trash and draw bounding boxes around it. Then, using array manipulations of the depth information, we transform the 2D coordinates to 3D in the camera link. This coordinate is then transformed to the world frame and input to a kinetic solver, which uses inverse kinematics to determine the joint angles required to reach the target coordinates using a velocity profile. Our system is visualized using RViz and Gazebo, providing a comprehensive solution for trash detection and handling.

Contributions

  • Forward and Inverse kinematics
  • Camera integration with YOLO
  • Pose transformations from 2D to 3D and geometrical verification
  • PANDA integration
  • World file creation and ROS integration
  • MoveIt integration
